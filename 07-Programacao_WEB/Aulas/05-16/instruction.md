# Estrutura de pastas
```bash
seu-projeto/
│
├── server.js
├── package.json
└── public/
    ├── index.html
    └── blog.html
```

# Comandos
### Se ainda não tiver o package.json, crie com:
- npm init -y

### Instalar o Express:
- npm install express
  - Isso criará a pasta node_modules/ e atualizará o package.json.

### Executar o servidor
- node server.js

### Testar no navegador
- http://localhost:3000/
